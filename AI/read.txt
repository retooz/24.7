횟수표시 기준 : 
# 오프셋 각도가 임계값보다 큰 경우
# 오프셋 각도가 임계값보다 작은 경우
# 아니면 이전 상태를 현재 상태로 업데이트




process 기준:
1. 포즈 랜드마크가 존재하는 경우 
-> 좌/우측 각도계산
    1-1. 오프셋 각도가 임계값보다 큰 경우 (정면 X)
    -> 휴식시간 계산 
    -> 프레임에 코, 어깨 좌표 및 프레임각도 표시
    1-2. 오프셋 각도가 임계값보다 작은 경우 (정면 O)
    -> 좌/우측 방향 체크하여 좌표 변경
    -> 특정 관절 수직각도 계산 및 점선 표시
        1-2-1. 현재 상태가 s1인 경우
        -> 올바른/잘못된 스쿼트 카운트
        -> 초기화
        1-2-2. 현재 상태가 s1이 아닌 경우
        -> 특정 관절 각도 계산해서 피드백 표시
